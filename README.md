# Graphics-Engine
So far just following https://vulkan.lunarg.com/doc/view/1.2.135.0/windows/tutorial/html/00-intro.html

## Requires
- VulkanSDK
- CMake
- Python 3.0+
- Visual Studio 2015, 2017, or 2019

## To build from root dir
`mkdir build`  
`cmake -B ./build`  
`cmake ./build`  
`Build.bat; ./build/Debug/Engine.exe`
